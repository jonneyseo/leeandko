# Lee & Ko Enterprises Inc.
## 로고 무드 탐색 — 네이비 & 차콜 버전

> 방향: B1 (신뢰·글로벌·전문) 유지 / 로고 무드만 다양하게 탐색  
> 각 무드는 슬로건, 심볼 아이디어, 타이포 방향, 느낌을 포함합니다.

---

## NAVY 계열 (딥네이비 베이스)

---

### N-1 | 앰퍼샌드 모노그램 (Ampersand Monogram)

**슬로건:** Connecting People. Connecting Worlds.

**컬러 팔레트**
| 역할 | 컬러명 | HEX |
|---|---|---|
| 메인 | 딥네이비 | `#0D1F3C` |
| 서브 | 미드나이트 블루 | `#1A3A5C` |
| 액센트 | 골드 | `#C9A84C` |
| 뉴트럴 | 쿨그레이 | `#8A9BB0` |
| 배경 | 오프화이트 | `#F7F8FA` |

**심볼 아이디어**
- `&` 기호 자체를 심볼로 활용
- 두꺼운 세리프 앰퍼샌드, 가운데 골드 포인트 라인 하나
- 심플하고 법인 느낌이 강함

**타이포 방향**
- 워드마크: **Playfair Display Bold** (세리프, 무게감 있음)
- 서브텍스트: Montserrat Regular

**무드 키워드:** 격식, 묵직함, 로펌/컨설팅, 클래식 프리미엄

---

### N-2 | 수평선 배지 (Horizon Badge)

**슬로건:** Built on Trust. Driven by People.

**컬러 팔레트**
| 역할 | 컬러명 | HEX |
|---|---|---|
| 메인 | 로얄네이비 | `#1B2F5E` |
| 서브 | 스틸블루 | `#3A5A8A` |
| 액센트 | 웜화이트 | `#F0EDE6` |
| 포인트 | 앰버 | `#D4890A` |
| 배경 | 크림 | `#FAFAF7` |

**심볼 아이디어**
- 가로로 긴 직사각형 배지 형태 (방패형 아님, 서류봉투형)
- 안에 `LEE & KO` 굵은 산세리프, 하단에 얇은 라인 하나로 `ENTERPRISES INC.` 구분
- 로고 자체가 배지처럼 독립적으로 사용 가능

**타이포 방향**
- 메인: **Inter Bold** (현대적 산세리프)
- 서브: Inter Light, letter-spacing 넓게

**무드 키워드:** 모던 법인, 캐나다 현지 기업 느낌, 깔끔, 뱃지처럼 직인 느낌

### N-3 | 이니셜 + 서클 씰 (Initial Seal)

**슬로건:** Your Trusted Partner in Canada.

**컬러 팔레트**
| 역할 | 컬러명 | HEX |
|---|---|---|
| 메인 | 딥네이비 | `#0D1F3C` |
| 서브 | 인디고 | `#2B4070` |
| 액센트 | 실버화이트 | `#E8ECF1` |
| 포인트 | 골드 | `#B8962E` |
| 배경 | 화이트 | `#FFFFFF` |

**심볼 아이디어**
- 원형 씰(seal) 안에 `L&K` 이니셜 대문자
- 씰 테두리에 얇은 선으로 `LEE & KO ENTERPRISES · VANCOUVER BC ·` 둘러싸기
- 정부/공기관 직인 느낌 → 신뢰감 극대화

**타이포 방향**
- 이니셜: **EB Garamond Bold** 또는 커스텀 레터링
- 둘레 텍스트: Montserrat SemiBold Small Caps

**무드 키워드:** 공식적, 인장/직인, 신뢰, 권위, 빈티지 기업 감성

### N-4 | 미니멀 워드마크 (Clean Wordmark)

**슬로건:** Lee & Ko — Where Business Meets Community.

**컬러 팔레트**
| 역할 | 컬러명 | HEX |
|---|---|---|
| 메인 | 딥네이비 | `#0D1F3C` |
| 포인트 | 스카이 블루 | `#4A8FC1` |
| 뉴트럴 | 미드그레이 | `#6B7C93` |
| 배경 | 퓨어화이트 | `#FFFFFF` |

**심볼 아이디어**

- `LEE & KO` 굵게, `ENTERPRISES INC.` 얇게, 두 줄 구성
- `&` 기호만 네이비보다 살짝 밝은 블루로 포인트

**타이포 방향**

- 서브: DM Sans Light

**무드 키워드:** 테크 스타트업, 글로벌, 심플, 스케일러블, 군더더기 없음

## CHARCOAL 계열 (차콜그레이 베이스)

---

### C-1 | 차콜 + 골드 프리미엄 (Charcoal Gold)

**슬로건:** Established in Vancouver. Built to Last.

**컬러 팔레트**
| 역할 | 컬러명 | HEX |
|---|---|---|
| 메인 | 차콜 | `#2C2C2C` |
| 서브 | 다크그레이 | `#3E3E3E` |
| 액센트 | 골드 | `#C9A84C` |
| 뉴트럴 | 미드그레이 | `#7A7A7A` |
| 배경 | 아이보리 | `#F5F3EF` |

**심볼 아이디어**
- 가는 사각 테두리 안에 `LEE & KO` (워드마크)
- 테두리 네 모서리에 골드 포인트 점
- 고급 호텔 / 프라이빗 클럽 느낌

**타이포 방향**
- 메인: **Cormorant Garamond SemiBold** (우아한 세리프)
- 서브: Raleway Light, letter-spacing 매우 넓게

**무드 키워드:** 프리미엄, 고급, 성숙, 조용한 권위, 모노톤 럭셔리

---

### C-2 | 차콜 모던 산세리프 (Charcoal Modern)

**슬로건:** Trusted. Versatile. Canadian.

**컬러 팔레트**
| 역할 | 컬러명 | HEX |
|---|---|---|
| 메인 | 다크차콜 | `#1E1E1E` |
| 서브 | 미디엄차콜 | `#3C3C3C` |
| 액센트 | 밝은화이트 | `#F2F2F2` |
| 포인트 | 테라코타 | `#C8553D` |
| 배경 | 오프화이트 | `#FAFAFA` |

**심볼 아이디어**
- 굵은 산세리프 `L` + `K` 두 글자를 슬래시(`/`)로 연결
- `LEE / KO` 형태 — 두 성씨가 대등하게 파트너십
- 테라코타 포인트가 따뜻함을 더함

**타이포 방향**
- 메인: **Space Grotesk Bold**
- 서브: Space Grotesk Regular

**무드 키워드:** 파트너십, 플랫한 모던, 에이전시 느낌, 젊고 진취적

---

### C-3 | 차콜 + 네이비 하이브리드 (Charcoal Navy Hybrid)

**슬로건:** Grounded in Vancouver. Growing Across Borders.

**컬러 팔레트**
| 역할 | 컬러명 | HEX |
|---|---|---|
| 메인 | 차콜 | `#272B33` |
| 서브 | 딥네이비 | `#1B2F5E` |
| 액센트 | 스카이블루 | `#5B9BD5` |
| 뉴트럴 | 라이트그레이 | `#B0B8C4` |
| 배경 | 크림화이트 | `#F8F8F6` |

**심볼 아이디어**
- 차콜 배경의 정사각형 심볼 블록 안에 네이비/화이트로 `L&K`
- 심볼 블록이 작은 아이콘처럼 독립 사용 가능 (명함, 파비콘 등)
- 산하 브랜드에 이 블록을 secondary 요소로 활용 가능

**타이포 방향**
- 메인: **Barlow SemiBold** (강하고 현대적)
- 서브: Barlow Regular

**무드 키워드:** 중립적, 확장성 높음, 산하 브랜드와 유연하게 결합, 실무적
